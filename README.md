### Activity的scheme跳转协议
+ android中的`scheme`是一种页面内跳转协议，是一种非常好的实现机制，通过定义自己的`scheme`协议，可以非常方便的跳转app中的各个页面，通过`scheme`协议，服务器可以定制化告诉app跳转到哪个页面，可以通过通知栏消息定制化跳转页面，可以通过H5页面定义链接跳转页面等；
